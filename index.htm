Blender

Blender is a free/open-source all-in-one 3d suite including • Video editor • Photo editor (to an extent, not very good) • 3d modeling • Sculpting • Rendering engines • Rigging/animating tools • Pretty much anything    Pretty much, it’s awesome, and it’s all i use for modeling/rendering. When I render out my projects I have to do it in .png format, so I can stop the render and start it again later. When i need to add them together as an animation instead of stills, I use Vegas Pro. Vegas Pro I don’t know a lot about this one, i just know it’s got the following trait. • Better video editor than blender
